# Smart Water Tracker

An Arduino-based hydration monitoring system that tracks water levels, reminds users to drink water, and monitors hydration inactivity.

## Features
- Real-time water level monitoring
- Hourly hydration reminders
- Drink tracking
- Low water alerts
- RTC-based scheduling
- OLED display support

## Components Used
- Arduino UNO
- HC-SR04 Ultrasonic Sensor
- DS3231 RTC Module
- Passive Buzzer
- OLED Display
